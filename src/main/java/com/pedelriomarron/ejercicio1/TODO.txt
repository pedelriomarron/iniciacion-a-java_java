Gestor de criticas unico independiente del objeto que lo invoque, (singleton)

metodos del gestor:
dar de alta a un usuario
dar de baja a un usuario
consultar los datos del usuario
actualizar los datos de un usuario

crear critica
consultar todas las criticas disponibles
borrar critica si el usuario es quien la escribio
votar la critica de otro
buscar criticas de un usuario registrado

Leer/ escribir datos en un fichero de texto plano, gestor de criticas se encarga de cargar el listado
